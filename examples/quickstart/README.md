# Quickstart Örnekleri

Bu klasör, MicroLog'a hızlıca başlamak için en basit örnekleri içerir.

## Örnekler

### 1. minimal_example.py
En minimal kullanım örneği. Sadece 3 satır kod ile logging başlatma.

**Kullanım:**
```bash
python examples/quickstart/minimal_example.py
```

### 2. hello_world.py
İlk adımlar için kapsamlı örnek. Tüm temel özellikleri gösterir.

**Kullanım:**
```bash
python examples/quickstart/hello_world.py
```

## Sonraki Adımlar

Quickstart örneklerini tamamladıktan sonra:

1. **Temel Kullanım**: `examples/basic/` klasörüne geçin
2. **Trace Context**: `examples/trace/` klasörüne geçin
3. **Web Framework**: `examples/web/` klasörüne geçin
